# Project 2 Recipes

# CRUD
In the CategoryListView, RecipeListView (when the filter is favorite or a category), edit mode in RecipeDetailsView, you can delete by swiping the recipe/detail you would like to remove. You can create a category in the edit mode in RecipeDetailsView or the CategoryListView. You can create a recipe in the RecipeListView. You can read, update, and delete the recipe in the edit mode of RecipeDetailsView. You can edit every aspect of the recipe besides the date added. The app icon was created by Gemini
